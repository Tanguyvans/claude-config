## Extraits AVANT (CTA section)

```
Le CTA par défaut est : "Voilà l'URL et à demain." C'est le format habituel du créateur, utilise-le sauf demande contraire.
```

## Extraits APRÈS (CTA section)

```
Le CTA est adapté au type de sujet :
- Repo / outil avec URL → "Voilà l'URL et à demain."
- News / tuto / opinion sans URL → CTA d'engagement ou pas de CTA
- Verbatim dicté → intégrer tel quel (orthographe seulement)
```
